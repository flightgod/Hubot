## Installation

In hubot project repo, run:

`npm install hubot-chucknorris --save`

Then add **hubot-chucknorris** to your `external-scripts.json`:

```json
[
  "hubot-chucknorris"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
