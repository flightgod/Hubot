# hubot-excuse

excuse

See [`src/excuse.coffee`](src/excuse.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-excuse --save`

Then add **hubot-excuse** to your `external-scripts.json`:

```json
[
  "hubot-excuse"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
