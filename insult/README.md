# hubot-insult

bot

See [`src/insult.coffee`](src/insult.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-insult --save`

Then add **hubot-insult** to your `external-scripts.json`:

```json
[
  "hubot-insult"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
